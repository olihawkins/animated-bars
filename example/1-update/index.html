1. Update

The simplest way to change the values shown in an animated chart is to call the chart's update method with new data. The new data should have the same keys as the initial data that was used to set up the chart, but with new values. Calling update will cause the chart to transition to the new values over an interval of time. The chart's default tranisition time is set with the configuration object's transitionTime property, but the the transition time can also be set for an individual transition using an optional second argument to update. The transition time is specified in milliseconds.
