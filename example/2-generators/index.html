2. Generators

Charts can be updated to show new values at fixed intervals using ES6 generators. Call a chart's run method with a generator to iterate throught a set of values. Each time the generator is called it should yield a new dataset containing new values for the same set of keys that were provided in the initial data that was used to set up the chart. The animated chart will transition between the values pulled from the generator using the transitionTime and pauseTime values set in the chart's configuration.

This is a bar chart running a generator. It is initialised with a dataset of zeros. The chart is then updated from a generator, which yields two alternating datasets, ten times in total. The final value the generator yields is the zero dataset again. Each of the datasets specifies an incremental delay to the timing at which each value is updated, creating a fan effect as each bar moves to its next value. Some of this chart's properties have been set using a custom configuration, including its colors, key axis location, and transition time.
